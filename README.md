## Setup

To run this project, install it locally using npm:

```
$ npm install
$ npm start or,
$ yarn start
```

## Features
* SignUp
* SignIn
* User Account
* Can edit user (name, avatar, birthdate, country/region)
* Global chat (content and image)
* Create post (content and image)
* Can add to post like and comment with image
